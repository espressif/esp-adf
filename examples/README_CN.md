# 例程

[English Version](./README.md)

本目录包含面向 ESP 平台的多媒体示例工程，覆盖音频与视频交互场景。这些示例提供了可参考的代码，便于在此基础上进行二次开发。

## 当前可用示例

| 示例 | 路径 | 说明 |
|---|---|---|
| 扣子 WebSocket 双向流式对话 | `examples/ai_agent/coze_ws_app` | 基于 ESP-GMF 的 Coze 语音对话示例。 |
| 百度 RTC 语音助手演示 | `examples/ai_agent/baidu_rtc/solutions/voice_assistant_app` | 基于 ESP-GMF 的百度 RTC 实时音视频助手示例。 |
